# Caporal AI

Sistema de soporte a la decisión para la optimización económica de dietas bovinas en entornos de datos escasos.

## Objetivo
Optimizar el costo de alimentación cumpliendo restricciones nutricionales (NRC) mediante un enfoque híbrido:
- Solver de optimización (programación lineal)
- Interfaz en lenguaje natural (RAG)

## Estructura del repositorio
- /backend-nestjs: API Gateway (NestJS)
- /ai-engine-python: microservicio de cálculo y IA (Python/FastAPI)
- /frontend-react: interfaz web (React)
- /docs: documentación técnica (arquitectura, notas)

## Tecnologías
NestJS, Python (FastAPI), React, PostgreSQL (planificado), Docker (planificado).
